# pentola_campaign_site

Single-file campaign website for **Mike Pentola**, candidate for **Nassau County Legislature, District 4**.

## Project Contents
- `index.html` — complete self-contained site (all CSS/JS inline)
- `robots.txt` — crawler directives
- `sitemap.xml` — homepage + section anchors sitemap
- `_headers` — Netlify security/cache headers
- `_redirects` — Netlify short-route redirects

## Deployment
Drag-and-drop these files into Netlify (no build step required).
